# 웹소설 서비스 클론코딩 프로젝트
- 개인 프로젝트 입니다.
- 백엔드 개발자로서, 테이블 설계, API설계, API 개발을 담당하였습니다.

## 프로젝트 목표

- 동시성 이슈를 대비한 로직을 설계해보자
: 공유된 자원에 동시에 여러 작업들이 접근하는 과정에서 동시성 이슈가 발생 할 수 있습니다. 이는 프로그램의 성능 저하를 일으키는 원인이 되므로 해당 이슈를 고려한 설계를 진행해야합니다.
